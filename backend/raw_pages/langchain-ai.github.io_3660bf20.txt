# Examples

The pages in this section provide end-to-end examples for the following topics:

## General

* [Template Applications](https://langchain-ai.github.io/langgraph/concepts/template_applications/): Create a LangGraph application from a template.
* [Agentic RAG](https://langchain-ai.github.io/langgraph/tutorials/rag/langgraph_agentic_rag/): Build a retrieval agent that can decide when to use a retriever tool.
* [Agent Supervisor](https://langchain-ai.github.io/langgraph/tutorials/multi_agent/agent_supervisor/): Build a supervisor agent that can manage a team of agents.
* [SQL agent](https://langchain-ai.github.io/langgraph/tutorials/sql/sql-agent/): Build a SQL agent that can execute SQL queries and return the results.
* [Prebuilt chat UI](https://langchain-ai.github.io/langgraph/agents/ui/): Use a prebuilt chat UI to interact with any LangGraph agent.
* [Graph runs in LangSmith](https://langchain-ai.github.io/langgraph/how-tos/run-id-langsmith/): Use LangSmith to track and analyze graph runs.
