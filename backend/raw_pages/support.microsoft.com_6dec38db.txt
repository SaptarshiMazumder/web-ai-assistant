Manage webinar recordings in Microsoft Teams

Review presentations from past webinars with webinar recordings, available in Microsoft Teams. Webinars are recorded automatically and available to download and publish as soon as the event ends.

## Record a webinar

By default, a webinar will automatically start recording when it begins. To start a webinar, join the webinar from your Teams calendar and select **Start meeting** in the meeting controls when you’re ready.

## Stop recording a webinar

By default, a webinar will start recording automatically when the event begins. To prevent a webinar from automatically recording, you'll need to turn off automatic recording in your webinar settings before the event starts. You can also stop recording during the webinar.

To turn off automatic recording before a webinar:

New calendarClassic calendar

1. Select **Calendar** ![Teams Calendar tab icon](/images/en-us/2975b9cc-8d8c-4cec-8585-63cadfca9b49)in Teams.
2. Select an event and select **Edit** ![Edit icon](/images/en-us/073d9d9a-0d84-402f-acdb-5765e790261f).
3. In **Details**,select **Meeting options** ![Settings button](/images/en-us/fe3fb52d-f1db-49e6-88cf-7d1ec1e8022e).
4. In **Recording and transcription** ![Record and transcribe icon](/images/en-us/225d74a7-b38f-4e2e-a237-c6bfb5242a5e), turn off the **Record and transcribe automatically** toggle.
5. Select **Apply**.

1. Select **Calendar**  ![Teams Calendar tab icon](/images/en-us/2975b9cc-8d8c-4cec-8585-63cadfca9b49) in Teams.
2. Right-click an event and select **Manage event** ![Edit icon](/images/en-us/073d9d9a-0d84-402f-acdb-5765e790261f).
3. Select **Meeting options** ![Settings icon](/images/en-us/e2a1f329-d95b-7d5e-181c-7c3e12fd9321).
4. In **Recording and transcription** ![Record and transcribe icon](/images/en-us/225d74a7-b38f-4e2e-a237-c6bfb5242a5e), turn off the **Record and transcribe automatically** toggle.
5. Select **Apply**.

​​​​​​​To stop recording during a webinar:

1. Select **More actions…** in your meeting controls.
2. Select **Record and transcribe** > **Stop recording**.

![Screenshot showing how to stop recording during a webinar](/images/en-us/d14a9ffa-d473-47c2-b8f8-f0d424e6d232)

## Publish a webinar recording

When you publish a webinar recording, registered attendees will automatically receive an [email with a link to the recording](/en-us/office/manage-webinar-emails-in-microsoft-teams-d0006848-f707-494f-b0a4-eeebcbc723be).

To publish a webinar recording:

New calendarClassic calendar

1. Select **Calendar**  ![Teams Calendar tab icon](/images/en-us/2975b9cc-8d8c-4cec-8585-63cadfca9b49) in Teams.
2. Select an event and select **Edit** ![Edit icon](/images/en-us/073d9d9a-0d84-402f-acdb-5765e790261f).
3. Select **Recordings**and find the recording you want to publish.
4. Select **Publish**.

   * If your recording was saved in OneDrive, select **Publish from OneDrive**  
       
      ![Screenshot showing users how to publish a webinar recording from One Drive](/images/en-us/15d628d7-02d3-4bc8-8f94-ad01ff6e89bf)

1. Select **Calendar**  ![Teams Calendar tab icon](/images/en-us/2975b9cc-8d8c-4cec-8585-63cadfca9b49) in Teams.
2. Right-click an event and select **Manage event** ![Edit icon](/images/en-us/073d9d9a-0d84-402f-acdb-5765e790261f).
3. Select **Recordings**and find the recording you want to publish.
4. Select **Publish**.

   * If your recording was saved in OneDrive, select **Publish from OneDrive**  
       
      ![Screenshot showing users how to publish a webinar recording from One Drive](/images/en-us/15d628d7-02d3-4bc8-8f94-ad01ff6e89bf)

## Access a webinar recording

To access a past webinar recording:

New calendarClassic calendar

1. Select **Calendar**  ![Teams Calendar tab icon](/images/en-us/2975b9cc-8d8c-4cec-8585-63cadfca9b49) in Teams.
2. Select an event and select **Edit** ![Edit icon](/images/en-us/073d9d9a-0d84-402f-acdb-5765e790261f).
3. Select **Recordings**.

1. Select **Calendar**  ![Teams Calendar tab icon](/images/en-us/2975b9cc-8d8c-4cec-8585-63cadfca9b49) in Teams.
2. Right-click an event and select **Manage event** ![Edit icon](/images/en-us/073d9d9a-0d84-402f-acdb-5765e790261f).
3. Select **Recordings**.

​​​​​​

You can also find webinar recordings and transcripts by selecting the past webinar in your Teams calendar and selecting the **Recap** tab. From there, you can watch the recording, search the transcript, and more. To learn more, see [Recap in Microsoft Teams](/en-us/office/recap-in-microsoft-teams-c2e3a0fe-504f-4b2c-bf85-504938f110ef).

## Download webinar transcripts

After a webinar ends, download the transcripts to save and review them later. If you stopped and restarted the transcription during the event, there will be multiple transcripts.

To download webinar transcripts:

New calendarClassic calendar

1. Select **Calendar**  ![Teams Calendar tab icon](/images/en-us/2975b9cc-8d8c-4cec-8585-63cadfca9b49) in Teams.
2. Select an event and select **Edit** ![Edit icon](/images/en-us/073d9d9a-0d84-402f-acdb-5765e790261f).
3. Select **Recordings**.
4. Select **Download transcripts** ![Teams download arrow icon](/images/en-us/4c7c01c7-af0f-49f5-aa24-4222f9ce9298).

   * If there are multiple transcripts, they’ll be downloaded as separate files.
   * Transcripts will download in both .docx and .vtt file types.

1. Select **Calendar**  ![Teams Calendar tab icon](/images/en-us/2975b9cc-8d8c-4cec-8585-63cadfca9b49) in Teams.
2. Right-click an event and select **Manage event** ![Edit icon](/images/en-us/073d9d9a-0d84-402f-acdb-5765e790261f).
3. Select **Recordings**.
4. Select **Download transcripts** ![Teams download arrow icon](/images/en-us/4c7c01c7-af0f-49f5-aa24-4222f9ce9298).

   * If there are multiple transcripts, they’ll be downloaded as separate files.
   * Transcripts will download in both .docx and .vtt file types.

## Unpublish a webinar recording

To unpublish a webinar recording, return to the recording and select **Unpublish**.

By default, published recordings will expire 30 days after you publish them. Extend them by choosing **Extend for 60 days** next to the published recording. After the new date passes, you’ll have to reupload and republish the file to keep sharing.  
  
![Screenshot showing how to unpublish a webinar recording](/images/en-us/2fb6c726-bab3-4204-be18-6b1f4bb1c01f)

You can also select **Download** to save the recording to your device and access anytime, anywhere.
