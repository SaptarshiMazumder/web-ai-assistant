You can see multiple calendars at the same time side-by-side or combined into a stacked overlay view to help you see availability across calendars.

Select a tab option below for the version of Outlook you're using. [What version of Outlook do I have?](https://support.microsoft.com/office/what-version-of-outlook-do-i-have-b3a9568c-edb5-42b9-9825-d48d82b2257c)

New OutlookClassic OutlookWeb

**Note:**If the steps under this **New Outlook** tab don't work, you may not be using new Outlook for Windows yet. Select [Classic Outlook](#picktab=classic_outlook) and follow those steps instead.

### Open a calendar in side-by-side view in new Outlook

1. On the navigation bar, select   **Calendar**.
2. From your Calendar list, select the other calendar(s) you want to view in addition to the default calendar.
3. To view the calendar side-by-side, in the **Arrange** group, select **Split view**.

   ![Screenshot of Calendar view with Split view option selected](/images/en-us/a70b49b8-00cd-4b51-94e3-4e64e861cf1c)

### Overlay the calendars in new Outlook

To view multiple calendars in a single calendar view, in the **Arrange** group, deselect **Split view**.  
  
The calendar events appear overlayed on the default calendar with their assigned color.

### Open a calendar in a new window in classic Outlook

1. In **Calendar**, in the **Navigation Pane**, right-click the name of the calendar that you want to view.
2. Select **Open in New Window**.

   ![Click Open in New Window on the right-click menu](/images/en-us/185ae168-e91b-4c13-af2c-6c37cd6f2746)

### Open a calendar in side-by-side view in classic Outlook

In **Calendar**, in the **Navigation Pane**, select the check box of another calendar that you want to view.

The calendar that you selected opens next to the calendar that already appears.

![Example of three calendars side by side](/images/en-us/f2c4915d-31e1-45b6-9474-f9a7cc3ed326)

### Overlay the calendars in classic Outlook

1. In **Calendar**, in the **Navigation Pane**, select the check box of another calendar that you want to view.

   The calendar that you selected opens next to calendar that is already displayed. Each successive calendar opens next to the one most recently opened.
2. On the calendar tab, select **View in Overlay Mode**.

   ![Click the View in Overlay Mode arrow](/images/en-us/39e5ad41-2d75-4d33-a6b2-b449d406f74a)

   That calendar is now in overlay mode with the one you opened first — usually your default calendar.
3. To add another calendar to the overlay, repeat step 2.

### Remove a calendar from the overlay stack in classic Outlook

1. In **Calendar**, on the tab of the calendar that you want to remove from the overlay stack, select **View in Side-by-Side Mode**.

   ![Click the View in Side-by-Side Mode arrow](/images/en-us/29d6e834-f565-44cc-ada5-07b12d87b859)

   The tab moves back to the right. The calendar that you selected no longer appears in overlay mode.
2. To remove a calendar from view, clear its check box in the Navigation Pane or select **Close** on the calendar tab.  
     
   [Top of Page](#)

### Open a calendar in side-by-side view in Outlook on the web or Outlook.com

1. On the navigation bar in the web version of Outlook, select   **Calendar**.
2. From your Calendar list, select the other calendar(s) you want to view in addition to your default calendar.
3. To view the calendar side-by-side, in the **Arrange** group, select **Split view**.

### Overlay the calendars in Outlook on the web or Outlook.com

To view multiple calendars in a single calendar view, in the **Arrange** group, deselect **Split view**.  
  
The calendar events appear overlayed on the default calendar with their assigned color.
